# Iterating WeChat Tech Posts

## When to use
结尾三版本、发布前质检、句子变锋利、数据复盘、下一篇怎么改。

## Inputs
- 文章主题/正文
- 发布数据（阅读量/完读率/分享/收藏/留言等）可选
- 目标偏好：涨粉/收藏/转发/留言

## Output expectations
- 4种结尾（高阶钩子）：价值延伸、互动提问、福利赠送、悬念预告。
- 发布前质检：结构收益、术语密度、**去机器味（AI 识别规避度）**、不确定事实清单。
- 段落锐化：改写版 + 真人感注入说明。
- 数据复盘：原因推断、下次改法、后续选题。

## Quality checks
- **去机器味优先**：严查书面化长句、空洞理论、生硬逻辑链。
- 强制注入：场景细节、主观情绪词、生活化衔接语。
- 改写需提升信息密度，同时确保“读起来像人在说话”。
- CTA 不要“营业员式硬推”，要利用价值与悬念吸引关注。
